Potenciación y Radicación en R

Pre-cálculo · Tecnología en Seguridad e Higiene Ocupacional

📖 Propiedades Fundamentales

Domina estas reglas y todo lo demás se vuelve mecánico. Cada propiedad tiene su fórmula, explicación y ejemplo.

💡
Consejo: No memorices, comprende. Todas las propiedades se derivan de la definición de potencia como multiplicación repetida.
🔶 Potenciación (8 propiedades)
Potenciación
P1 · Producto de potencias de igual base
$$a^{m} \cdot a^{n} = a^{m+n}$$
Cuando multiplicas potencias con la misma base, se mantiene la base y se suman los exponentes.
✅ $2^3 \cdot 2^4 = 2^{3+4} = 2^7 = 128$
Potenciación
P2 · Cociente de potencias de igual base
$$\frac{a^{m}}{a^{n}} = a^{m-n} \qquad (a \neq 0)$$
Al dividir potencias con la misma base, se mantiene la base y se restan los exponentes.
✅ $\dfrac{5^6}{5^2} = 5^{6-2} = 5^4 = 625$
Potenciación
P3 · Potencia de una potencia
$$\left(a^{m}\right)^{n} = a^{m \cdot n}$$
Cuando elevas una potencia a otro exponente, se mantiene la base y se multiplican los exponentes.
✅ $(3^2)^4 = 3^{2 \cdot 4} = 3^8 = 6561$
Potenciación
P4 · Potencia de un producto
$$(a \cdot b)^{n} = a^{n} \cdot b^{n}$$
El exponente se distribuye a cada factor del producto.
✅ $(2 \cdot 3)^4 = 2^4 \cdot 3^4 = 16 \cdot 81 = 1296$
Potenciación
P5 · Potencia de un cociente
$$\left(\frac{a}{b}\right)^{n} = \frac{a^{n}}{b^{n}} \qquad (b \neq 0)$$
El exponente se distribuye al numerador y al denominador.
✅ $\left(\dfrac{4}{3}\right)^2 = \dfrac{16}{9}$
Potenciación
P6 · Exponente cero
$$a^{0} = 1 \qquad (a \neq 0)$$
Toda base distinta de cero elevada a la cero es 1. Se deduce de $\frac{a^n}{a^n} = a^{n-n} = a^0 = 1$.
✅ $7^0 = 1$, $\;(-5)^0 = 1$, $\;x^0 = 1$
Potenciación
P7 · Exponente negativo
$$a^{-n} = \frac{1}{a^{n}} \qquad (a \neq 0)$$
Un exponente negativo indica el inverso multiplicativo. "Pasar al otro lado de la fracción cambia el signo del exponente."
✅ $2^{-3} = \dfrac{1}{2^3} = \dfrac{1}{8}$
Potenciación
P8 · Base negativa
$$(-a)^n = \begin{cases} a^n & \text{si } n \text{ es par} \\ -a^n & \text{si } n \text{ es impar} \end{cases}$$
El signo depende de la paridad: par → positivo, impar → negativo.
✅ $(-2)^4 = 16$  |  $(-2)^5 = -32$
🔷 Radicación (6 propiedades)
Radicación
R1 · Definición como exponente fraccionario
$$\sqrt[n]{a} = a^{\frac{1}{n}}$$
Todo radical se puede escribir como potencia con exponente fraccionario. Es la conexión clave entre radicación y potenciación.
✅ $\sqrt[3]{8} = 8^{1/3} = 2$
Radicación
R2 · Raíz de un producto
$$\sqrt[n]{a \cdot b} = \sqrt[n]{a} \cdot \sqrt[n]{b}$$
La raíz de un producto es el producto de las raíces. Puedes "separar" factores dentro del radical.
✅ $\sqrt{36 \cdot 25} = 6 \cdot 5 = 30$
Radicación
R3 · Raíz de un cociente
$$\sqrt[n]{\frac{a}{b}} = \frac{\sqrt[n]{a}}{\sqrt[n]{b}} \qquad (b \neq 0)$$
La raíz de un cociente es igual al cociente de las raíces.
✅ $\sqrt{\dfrac{49}{16}} = \dfrac{7}{4}$
Radicación
R4 · Raíz de una potencia
$$\sqrt[n]{a^{m}} = a^{\frac{m}{n}}$$
Para "sacar" una potencia del radical, divide el exponente entre el índice.
✅ $\sqrt{x^6} = x^{6/2} = x^3$
Radicación
R5 · Radical de un radical
$$\sqrt[m]{\sqrt[n]{a}} = \sqrt[m \cdot n]{a} = a^{\frac{1}{mn}}$$
Un radical dentro de otro equivale a un solo radical cuyo índice es el producto de ambos.
✅ $\sqrt{\sqrt[3]{64}} = \sqrt[6]{64} = 2$
Radicación
R6 · Simplificación de índice y exponente
$$\sqrt[n \cdot k]{a^{m \cdot k}} = \sqrt[n]{a^{m}}$$
Se puede simplificar dividiendo índice y exponente por un factor común.
✅ $\sqrt[6]{x^4} = \sqrt[3]{x^2}$ (dividiendo entre 2)
